# Sitio Web de Biografía (React)

Aplicación web desarrollada con React que presenta información biográfica mediante una interfaz interactiva y organizada en componentes.

##  Descripción
Este proyecto consiste en una página web que muestra información biográfica estructurada en distintas secciones, utilizando componentes reutilizables en React.

## Funcionalidades
- Visualización de información biográfica
- Navegación entre secciones
- Uso de componentes reutilizables
- Interfaz desarrollada con HTML, CSS y JavaScript

## Tecnologías utilizadas
- React
- JavaScript
- HTML5
- CSS3
